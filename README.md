CourseGrab 是一个专为北京理工大学学生设计的辅助选课系统。它通过模拟浏览器请求帮助用户更高效地管理课程选择，支持实时状态监控、智能任务队列和即时通知。

## ✨ 主要特性

- **🔍 智能课程检索**
  - 支持多条件（关键字、院系）搜索
  - 实时显示课程余量、时间冲突等信息
  - 一键加入抢课队列

- **🚀 自动化抢课引擎**
  - **多任务并发**：支持同时监控多个课程
  - **智能冲突处理**：同门课程（不同班级）抢到其一后自动取消其他任务
  - **状态验证**：抢课成功后二次查证，确保真实中签

- **📊 可视化仪表盘**
  - **实时数据**：展示队列状态、成功/失败统计
  - **我的课表**：同步学校教务系统课表，支持按学期切换查看
  - **课程列表**：详细展示已选课程状态（正式选中/已中签/待抽签/未中签），包含学分统计
  - **历史记录**：清晰展示抢课任务的历史结果

- **📱 现代化交互**
  - **响应式设计**：完美支持 PC 和移动端访问
  - **实时反馈**：WebSocket 推送抢课进度和结果
  - **操作便捷**：支持一键退课（带二次确认）、一键清空队列/历史

- **🔔 消息通知**
  - **多渠道支持**：集成 SMTP 邮件通知和企业微信机器人通知
  - **即时触达**：抢课成功后第一时间发送详细通知（课程名、时间、地点）

- **🔐 安全认证**
  - 使用 mitmproxy 代理抓包获取 Cookie，无需保存用户账号密码（待完善）
  - 自动维护 CSRF Token，处理会话过期自动重试

## 🛠️ 技术栈

- **后端**：Python 3.10+, FastAPI, SQLAlchemy (SQLite), HTTPX (Async), mitmproxy
- **前端**：HTML5, CSS3 (Modern Variables), Vanilla JavaScript (ES6+)
- **通信**：WebSocket, RESTful API

## 🚀 快速开始

### 1. 环境准备

确保已安装 Python 3.10 或更高版本。

```bash
# 克隆项目
git clone https://github.com/LingRuo-BIT/LingRuo-BIT-CourseGrab.git
cd CourseGrab

# 安装依赖
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python run.py
```

### 3. 获取凭证

您可以通过以下方式获取登录凭证。

**手动获取 Cookie**

1. 在浏览器中登录选课系统 `xk.bit.edu.cn`。
2. 按 `F12` 打开开发者工具，切换到"网络" (Network) 标签。
3. 刷新页面，点击任意请求 (如 `xsxkHome`)。
4. 在"请求标头" (Request Headers) 中找到 `Cookie` 字段。
5. 复制**完整**的 Cookie 值（形如 `EMAP_LANG=zh; GS_SESSIONID=...`）。
6. 在 CourseGrab 网页底部的"登录凭证配置"区域粘贴并保存。
